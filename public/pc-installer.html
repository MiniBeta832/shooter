Bot Breaker 3D Installer

Esta pagina descarga el instalador para Windows. El instalador es un wizard (siguiente, siguiente, instalar) y deja el juego instalado en tu PC.

Descargar Setup.exe Ir al release de GitHub Volver al juego

Descarga directa desde GitHub Releases.

  1. Descarga BotBreakerSetup.exe.
  2. Abre el archivo y acepta permisos de Windows.
  3. Completa el wizard de instalacion.
  4. Abre Bot Breaker 3D desde el acceso directo creado.

Si aun no existe el Setup.exe, generarlo en esta carpeta con crear-setup-pc.bat.
